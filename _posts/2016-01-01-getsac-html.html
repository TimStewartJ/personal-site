A simple 2D game that won 1st in Game Development at SacHacks III.

About the Project

getsac.online is a simple 2D game developed for SacHacks III. The game was awarded 1st place in the Game Development category.

Features

The game includes various features such as:

  • Engaging gameplay mechanics
  • High-quality graphics
  • Multiplayer support

Conclusion

getsac.online is a testament to the creativity and skill of its developers, showcasing what can be achieved in a short time frame.
