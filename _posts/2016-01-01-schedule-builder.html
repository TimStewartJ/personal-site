A tool that will automatically generate schedules given an input of classes.

About the Project

The UC Davis Automatic Schedule Builder is a tool designed to help students generate schedules based on their class inputs. It simplifies the process of schedule creation and ensures that there are no conflicts.

Features

The tool includes various features such as:

  • Automatic schedule generation
  • Conflict detection
  • User-friendly interface

Conclusion

This project is a valuable resource for students at UC Davis, making the process of schedule creation much more efficient and stress-free.
